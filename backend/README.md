# Ideen Ingest Channel Backend

SSH-zugänglicher Drag-and-Drop Eingangskanal für GBrain Ideen-System.

## Installation

```bash
python3 -m venv venv
source venv/bin/activate
pip install -e .
```

## Running

```bash
source venv/bin/activate
python -m src.main
```

The server will start on `http://0.0.0.0:8000`
