# Development Workflow Skill

Koordiniert Development-Aufgaben innerhalb von WUPHF und kann bei Bedarf Open Design für UI-Komponenten aufrufen.

## Trigger

Wird aufgerufen bei:
- Development-Aufgaben (Code, Architektur, Debugging)
- Software-Integration
- API-Entwicklung

## Verwendung

Agenten können diesen Skill verwenden, um komplexe Development-Projekte zu koordinieren:

```
Entwickle eine REST-API für unser User-Management mit Authentication
```

Der Skill analysiert die Aufgabe und kann bei Bedarf UI-Komponenten an Open Design delegieren.
