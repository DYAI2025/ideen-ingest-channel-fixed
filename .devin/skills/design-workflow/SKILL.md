# Design Workflow Skill

Leitet Design-Aufgaben automatisch an Open Design weiter.

## Trigger

Wird aufgerufen bei:
- Design-Anfragen (UI/UX, Grafiken, Präsentationen)
- Visualisierungsaufgaben
- Branding und Marketing-Material

## Verwendung

Agenten können diesen Skill verwenden, um Design-Aufgaben an Open Design zu delegieren:

```
Erstelle ein modernes Dashboard-Design für unsere Analytics-Plattform
```

Der Skill leitet die Aufgabe an den Workflow-Orchestrator, der sie an Open Design weiterleitet.
